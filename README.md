# odin-recipes

## Project Description
A recipe website built to practice foundational HTML skills for The Odin Project.

The site consists of a main index page linking to multiple recipe pages. The focus is on structural HTML; styling with CSS will be added in future lessons.

## Skills

* Structure multiple web pages using HTML.
* Use semantic HTML elements. 
* Create nested navigation to link homepage and sub-pages.
* Organize a project environment by managing files within and across directories.
* Use Git, GitHub, VS Code, and Markdown.
